# job-queue

An experiment for exploring a small job queue implementation in TypeScript using Bun.

This project is intentionally self-contained inside the broader experiments repository. The implementation and docs are expected to change as the experiment evolves.

## Setup

To install dependencies:

```bash
bun install
```

## Run

To run:

```bash
bun run index.ts
```

## Notes

- Runtime: Bun
- Language: TypeScript
- Scope: Local experiment, not a packaged library

This project was created using `bun init` in bun v1.3.14. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
